Design a modern tablet-first POS system UI for a coffee shop. The interface should use a clean split-layout design optimized for fast cashier workflows and touch interaction.

The left side should contain:

- category tabs at the top
- large product cards arranged in a responsive grid
- pastel category-based colors
- product name and price clearly visible
- large touch-friendly buttons
- minimal clutter

The right side should contain:

- a persistent current order panel
- cart items with quantity controls
- subtotal, tax, and total summary
- large checkout buttons for card and cash payments
- clean spacing and strong visual hierarchy

The overall interface should feel:

- lightweight
- operational
- fast
- modern
- minimal
- tablet-optimized
- low friction

Avoid:

- enterprise dashboard complexity
- excessive sidebars
- dense tables
- tiny buttons
- unnecessary charts
- overdesigned UI elements

Target Device:

- 11–12 inch Android tablet
- landscape orientation
- touch-first usage

Style References:

- modern café POS systems
- kiosk interfaces
- clean operational dashboards
- soft pastel color palette
- rounded cards and buttons
- high readability
